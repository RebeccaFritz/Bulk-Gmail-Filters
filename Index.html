Results
Key Value Description
from: email@example.com Filter by sender email address
to: email@example.com Filter by recipient email address
subject: any text Filter by email subject line
query: any text or phrase Filter by email content containing specific words/phrases
negatedQuery: any text or phrase Filter by email content NOT containing specific words/phrases
hasAttachment: true or false true to filter only emails with attachments
excludeChats: true or false true to exclude chat messages from filter
size: any number of bytes (e.g. 1000000) Used with sizeComparison: to filter by attachment size
sizeComparison: larger or smaller Used with size: to filter by attachment size
label: any label (to set multiple labels use the format [label1, label2, etc]) Apply a specific label to matching emails
skipInbox: true or false Archive matching emails instead of keeping in inbox
markAsRead: true or false Automatically mark matching emails as read
star: true or false Star matching emails
markImportant: true or false Always mark matching emails as important
neverMarkImportant: true or false Prevent matching emails from being marked important
neverSpam: true or false Prevent matching emails from going to spam
forwardTo: email@example.com Forward matching emails to the provided email address
delete: true or false Automatically delete matching emails
category: Promotions, Social, Updates, Forums, Primary Categorize matching emails
backfill: true or false Apply filter to all matching conversations (App only! Not a filter setting)
